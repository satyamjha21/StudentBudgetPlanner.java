# 🎓 Student Budget Planner (Java)

A simple console-based **Student Budget Planner** built using Java.
This application helps college students manage their monthly budget by tracking expenses and calculating remaining balance.

---

## 📌 Features

* Set monthly budget
* Add expenses with category
* View all expenses
* Calculate total spending
* Show remaining budget
* Budget warning when exceeded
* Menu-driven console interface

---

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Loops & Conditional Statements
* VS Code (IDE)

## ▶️ How to Run

### Step 1 — Compile

```

```
